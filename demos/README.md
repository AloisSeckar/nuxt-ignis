# Nuxt Ignis Demos

This folder contains various implementations of `nuxt-ignis` module. Those mini-apps are used to check new features, as an input for automated tests and for reference of this project's capabilities.

## Available demos

01. Default features
    - a) `01-01-default` - default setting with no `.env` modifications
    - b) `01-02-bare` - default features disabled - smallest possible setup
    - c) `01-03-full` - all features enabled - largest possible setup
02. UI features
    - a) `02-01-nuxt-ui` - Nuxt UI module enabled
    - b) `02-02-tailwind` - Tailwind CSS module enabled
    - c) `02-03-open-props` - Open Props enabled
    - d) `02-04-charts` - Nuxt Charts enabled
03. DB features
    - a) `03-01-neon` - Neon module enabled
    - b) `03-02-supabase` - Supabase module enabled
04. Form features
    - a) `04-01-vueform` - Vueform module enabled
    - b) `04-02-formkit` - FormKit module enabled
05. Validation features
    - a) `05-01-valibot` - Valibot module enabled
    - b) `05-02-zod` - Zod module enabled
06. Content features
    - a) `06-01-content` - Nuxt Content module enabled
    - b) `06-02-content-tailwind` - Nuxt Content module enabled along with Tailwind CSS
    - c) `06-03-content-pslo` - Nuxt Content module enabled with pslo integration
    - d) `06-04-i18n` - I18N module enabled
    - e) `06-05-social` - Nuxt Social Share module enabled
    - f) `06-06-seo` - Nuxt SEO module enabled
    - g) `06-07-seo-ssr` - Nuxt SEO module enabled with SSR
07. Utils features
    - a) `07-01-equipment` - Vue Equipment module enabled
    - b) `07-02-regexp` - Magic Regexp module enabled
